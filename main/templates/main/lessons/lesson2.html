Выделение объектов и работа с 3D-курсором в сцене Blender

Blender предоставляет пользователям мощные инструменты для точного взаимодействия с 3D-объектами, а также удобные средства позиционирования — такие как 3D-курсор. Умение правильно выделять элементы сцены и управлять 3D-курcором является одной из базовых и важнейших составляющих в освоении Blender.


Выделение объектов

Выделение объектов — это основной способ взаимодействия со сценой. Любая трансформация (перемещение, масштабирование, вращение), применение модификаторов, работа в различных режимах начинается с выбора нужного объекта.

Основные способы выделения:

№ Способ выделения Описание Изображение
1 ЛКМ (левая кнопка мыши) Выбор объекта одним щелчком.В Blender для выделения объектов по умолчанию используется левая кнопка мыши. Когда пользователь щёлкает по объекту, тот подсвечивается оранжевой рамкой, а также становится активным — его центр отображается в виде маленькой точки, а гизмо трансформации позволяет выполнять дальнейшие действия. Если ранее был выделен другой объект, он автоматически теряет выделение.
2 Shift + ЛКМ Для выделения нескольких объектов одновременно используется комбинация Shift и левой кнопки мыши. Это позволяет поочерёдно добавлять объекты к текущей группе выделения или, наоборот, снимать выделение с ранее выделенного объекта.
3 Выделение рамкой (Box Select) Если нужно выделить сразу несколько объектов в области сцены, удобно использовать рамочное выделение, которое вызывается клавишей B. После её нажатия появляется рамка, и, протянув её по сцене, можно захватить все объекты, попавшие внутрь.
4 Круговое выделение (Circle Select) Для более точного управления подойдёт круговое выделение, активируемое клавишей C. На экране появляется круг, радиус которого регулируется колесиком мыши. Пользователь может выделять объекты, просто кликая по ним внутри круга. Выход из режима кругового выделения происходит нажатием правой кнопки мыши или клавиши Esc.
5 Лассо-выделение (Lasso Select) Также в Blender предусмотрено лассо-выделение. Оно активируется комбинацией Ctrl и правой кнопки мыши. В этом режиме пользователь может нарисовать произвольную фигуру, ограничивающую область выделения.

После выделения объект становится активным, и именно с ним будут производиться все действия по умолчанию. Активный объект отображается жёлтой рамкой (если выделено несколько).

Объекты можно выделять и в редакторе Outliner | Аутлайнер. Здесь же их можно переименовывать, скрывать видимость, сортировать по коллекциям и др.

Работа с 3D-курcором

3D-курсор — это вспомогательный инструмент, предназначенный для точного позиционирования объектов, начала трансформаций, добавления новых объектов и установки точки вращения.
По умолчанию, при добавлении нового объекта в сцену, он появляется в позиции 3D-курсора.

Как установить 3D-курсор:

  • Shift + Правая кнопка мыши (ПКМ) — установить курсор в любое место сцены.
  • В последних версиях Blender используется Shift + ЛКМ.
  • очно задать координаты можно во вкладке View → 3D Cursor (в N-панели).

Назначение 3D-курсора:

  1. Добавление нового объекта в сцену (он появится в позиции курсора).
  2. Поворот объектов вокруг курсора, если выбран режим поворота 3D Cursor в верхней панели Pivot Point.
  3. Упрощение точного выравнивания и позиционирования объектов.
  4. Быстрое смещение объектов на позицию курсора с помощью команды Snap → Selection to Cursor (Shift + S).

Gizmo трансформации

После выделения объекта в сцене Blender автоматически отображает гизмо трансформации — это визуальные стрелки, дуги и квадраты, расположенные на объекте, которые позволяют:

  1. Перемещать (Move) — стрелки направлены по осям X, Y, Z.
  2. Вращать (Rotate) — дуги вокруг объекта позволяют вращать по выбранной оси.
  3. Масштабировать (Scale) объект — кубики на концах направлений позволяют увеличивать или уменьшать объект.

! Если по умолчанию Гизмо не появляется при выделении обьекта вы можете включить вручную эту настройку в правом верхнем углу окна 3D Viewport:

Центры вращения (Pivot Point)

Центр вращения определяет, вокруг какой точки вращается объект:

  • Active Element: вращение вокруг активного элемента (последнего выбранного объекта).
  • Median Point: вращение вокруг среднего значения координат всех выделенных объектов.
  • 3D Cursor: вращение вокруг положения 3D курсора.
  • Individual Origins: каждый объект вращается вокруг своей локальной оси.
  • Bounding Box Center: вращение вокруг центра ограничивающего прямоугольника выделенных объектов.

Чтобы сменить Pivot Point:

  1. Выберите объекты.
  2. В верхнем меню 3D Viewport найдите меню Pivot Point (или нажмите . (точка) на клавиатуре).
  3. Выберите нужный режим.
