Источники света в Blender. Настройки мира

Освещение играет важнейшую роль в создании реалистичных и выразительных 3D-сцен. Если удалить источник света на рендере мы получим черный экран. В Blender существует несколько типов источников света, каждый из которых имеет свои особенности и используется для разных задач. В этом уроке мы изучим основные виды света и научимся их настраивать.

Основные типы источников света

  • Point Light — точечный свет.
  • Sun Light — имитация солнечного света.
  • Spot Light — прожектор, направленный конусом.
  • Area Light — свет, исходящий от поверхности.

Point Light (Точечный свет)

Point Light испускает свет во всех направлениях из одной точки, подобно лампочке. Он хорошо подходит для подсветки небольших объектов или создания атмосферных эффектов.

Sun Light (Солнечный свет)

Sun Light создаёт параллельные лучи света, как от солнца. Он используется для освещения больших открытых сцен, где важна равномерность света и теней.

Spot Light (Прожектор)

Spot Light испускает свет в форме конуса. Это отличный вариант для создания сценических эффектов или акцентного освещения.

Area Light (Плоский источник света)

Area Light излучает свет с заданной поверхности. Он используется для мягкого равномерного освещения, например, имитации света от окна или студийного оборудования.

Как добавить источник света

  1. Нажмите Shift + A → Light.
  2. Выберите тип источника света из списка.
  3. Переместите, поверните и настройте параметры в панели Properties → Object Data Properties (иконка лампочки).

Основные параметры света

  • Power — мощность света (измеряется в ваттах).
  • Color — цвет освещения.
  • Size / Radius — влияет на мягкость теней.
  • Spot Size (для Spot Light) — угол распространения света.
  • Strength (для Sun Light) — интенсивность света.

Практический пример освещения сцены

  1. Создайте куб (Shift + A → Mesh → Cube).
  2. Добавьте источник света Sun Light.
  3. Настройте угол наклона источника, чтобы получить мягкие длинные тени.
  4. Переключитесь в режим рендера (Render Preview) и сохраните результат.

Советы по работе со светом

  • Комбинируйте разные источники света для создания реалистичных сцен.
  • Регулируйте мощность и цвет света в зависимости от настроения сцены.
  • Для мягких теней увеличивайте размер источника.
  • Не забудьте про фон (World Settings), который тоже может давать свет.

Окружающая среда (мир) в Blender задаёт фон сцены и влияет на общее освещение объектов. Правильная настройка мира позволяет создать атмосферу сцены, добавить реалистичное освещение или установить фон для рендеринга.
