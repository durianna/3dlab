Дополнительные инструменты редактирования в Blender

Практическая работа №10

Цель работы

Научиться использовать различные инструменты редактирования при моделировании обьекта в Blender

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Удалить стартовый куб Выделите куб → нажмите X → выберите Delete.
2 Добавить плоскость Shift + A → Mesh → Plane.
3 Перейти в режим редактирования Выделите плоскость → нажмите Tab → выберите Edit Mode.
4 Масштабировать плоскость Нажмите S → X → 2, затем S → Y → 1.5 — придайте плоскости форму столешницы.
5 Сделать плоскость объёмной Выделите всю плоскость (A) → нажмите E → потяните вверх (например, на 0.2).
6 Разрезать объект по оси X Используем инструмент Loop Cut для создания разрезов на гранях столешницы. Ctrl + R → наведите на ось → После того как появится желтая линия нажмите ЛКМ, затем двигайте линию ближе с краю столешницы, для формирования будущих ножек стола. Проделайте это с двух сторон. Для удобства переключитесь на вид сверху(7)
7 Разрезать объект по оси Y Повторите действия п.6, но разрезайте по другой оси.
8 Выделить угловые грани Переключитесь в Face Select (3) → выделите четыре угловые грани снизу с помощью Shift.
9 Экструдировать ножки вниз Нажмите E — вытяните ножки вниз.
9.1 Удаляем лишние грани Переключитесь на вид сверху(7), выделите столешницу. Нажмите правую кнопку мыши - Dissolve Faces
10 Выделить столешницу Включите режим Toogle X-Ray в правой верхней части Viewport. Выделите верхнюю часть столешницы(режим отображения вершин)
11 Применить фаску к краям (Bevel) Ctrl + B → потяните мышью → прокрутите колесо мыши, чтобы добавить сегменты фаски.
12 Добавить обьекты Добавьте на стол дополнительные обьекты, например тарелки или кружки из предыдущей практической работы
13 Переключиться в рендеринг В правом верхнем углу 3D Viewport нажмите Z → Rendered, либо выберите иконку шарика.
14 Настроить камеру Shift + A → Camera, нажмите 0 на NumPad → G и R для позиционирования.
15 Сделать финальный рендер Нажмите F12 для рендера сцены.
16 Сохранить изображение В окне рендера: Image → Save As.



Ожидаемый результат:

  • Переключение между различными режимами выделения (вершины, рёбра, грани).
  • Перемещение, масштабирование и вращение элементов меша.
  • Использование инструментов Extrude, LoopCut, Bevel

Практическая работа №11

Цель работы

3D-модель карандаша, созданная с использованием цилиндра, инструментов Loop Cut, Extrude, Scale и Bevel.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Создайте новый проект и удалите куб Создайте новый файл: File → New → General. Удалите стартовый куб: выделите его → нажмите X → Delete.
2 Добавьте цилиндр Shift + A → Mesh → Cylinder.
3 Перейдите в режим редактирования Выделите цилиндр → нажмите клавишу Tab → Edit Mode.
4 Масштабируйте цилиндр по длине Нажмите S → Z → 2 или 3, чтобы вытянуть цилиндр по вертикальной оси (основа карандаша).
5 Создайте Loop Cut на цилиндре Используйте инструмент Loop Cut (Ctrl + R). Наведите на цилиндр и прокрутите колесико мыши, чтобы создать несколько горизонтальных разрезов (3-4). Затем подтвердите разрезы кликом мыши (ЛКМ → ЛКМ).
6 Сформируйте наконечник карандаша Выделите верхнюю грань цилиндра (Face Select → выделите верхнюю грань). Нажмите E → вытяните немного вверх, затем нажмите S → уменьшите масштаб, формируя острую форму наконечника. Повторите эти действия несколько раз, постепенно уменьшая масштаб для создания заострённой формы.
7 Создайте сужение карандаша снизу Выделите нижнюю грань, нажмите E → вытяните вниз совсем немного, затем S → немного уменьшите масштаб, чтобы сформировать область стержня карандаша.
8 Создайте Bevel (скругление) рёбер Перейдите в режим выделения рёбер (Edge Select), нажмите Toggle X-Ray, выделите рёбра на гранях наконечника карандаша и нижнего края, затем нажмите Ctrl + B и аккуратно двигайте мышью для скругления. Подтвердите действие кликом (ЛКМ). По желанию добавьте детали, например - ластик
9 Настройте освещение сцены Выйдите из режима редактирования (Tab), добавьте источник света: Shift + A → Light → Point. Переместите источник света вверх и чуть в сторону, чтобы карандаш был хорошо освещён.
10 Выполните рендер карандаша Перейдите во вкладку Render Properties и выберите движок рендера (Cycles или Eevee). Затем нажмите F12 для запуска рендера. Сохраните результат: Image → Save As.



Ожидаемый результат:

  • 3D-модель карандаша, созданная с использованием цилиндра, инструментов Loop Cut, Extrude, Scale и Bevel.

Вопросы для самоконтроля:

  1. Какой инструмент используется для создания дополнительных разрезов на объекте?
  2. С помощью какой комбинации клавиш можно выполнить скругление ребер модели?
  3. Как отменить последнее действие в Blender, если была допущена ошибка?
  4. Как изменить количество разрезов при использовании инструмента Loop Cut?

Практическая работа №12

Цель работы

Сформировать модель стула с использованием базовых инструментов редактирования меша и точного позиционирования элементов.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Удалить стартовый куб Выделите куб → нажмите X → выберите Delete.
2 Добавить плоскость (сидение) Нажмите Shift + A → Mesh → Plane.
3 Придать объём сидушке Tab → Edit Mode → A → E → вытяните вверх на 0.1–0.2 ед.
4 Добавить разрез (Loop Cut) Ctrl + R → прокрутите колесо → создайте 1 разрез и переместите ближе к краю. Таким же образом сделайте еще 3 разреза, у вас должны появится 4 угловые грани для будущих ножек стула.
5 Выделить зоны для ножек Face Select (3) → выделите 4 угловые грани снизу
6 Экструдировать ножки E → потянуть вниз на 1.5–2 ед.
7 Сделать фаски (Bevel) Выделите вертикальные грани ножек → Ctrl + B → двигайте мышь
8 Добавить перекладину между ножками Добавим новый обьект для перекладины. Shift + A → Add Cube. Затем масштабируем куб так чтобы он стал похож на перекладину стула с помощью → S, затем переместим его с помощью G → разместите между передними ножками
9 Включить Snap для выравнивания Включите значок Snap (магнит) → режим Face или Vertex → выровняйте. Если вы случайно потеряли выделение с добавленного обьекта вы можете вернуть его нажав CTRL+Z, выделение в Blender так же считается действием. После использования выключите Snap повторным нажатием на магнитик.
10 Добавить вторую перекладину Shift + D → переместить к задним ножкам
11 Создать спинку стула Выделите 2 угловые грани сидения сверху, затем экструдируйте их → E вверх, это будет основа для спинки
12 Добавить перекладины для спинки Shift + A → Cube → масштаб → переместить между опорами спинки, аналогично с перекладинами для ножек стула создайте перекладины для спинки. Здесь вы также можете использовать "магнит" для того, чтобы ровно разместить их.
13 Выполните рендер Перейдите во вкладку Render Properties и выберите движок рендера (Cycles или Eevee). Затем нажмите F12 для запуска рендера. Сохраните результат: Image → Save As.



Ожидаемый результат:

  • Модель стула с четырьмя ножками, двумя нижними перекладинами и спинкой.
  • Применение Loop Cut, Extrude, Bevel и выравнивания через Snap.

Вопросы для самоконтроля:

  1. Для чего используется инструмент Bevel?
  2. Как включить режим выравнивания Snap и в чём его польза?
