Дополнительные инструменты редактирования в Blender

Практическая работа №

Цель работы

Научиться использовать различные инструменты редактирования при моделировании обьекта в Blender

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Удалить стартовый куб Выделите куб → нажмите X → выберите Delete.
2 Добавить плоскость Shift + A → Mesh → Plane.
3 Перейти в режим редактирования Выделите плоскость → нажмите Tab → выберите Edit Mode.
4 Масштабировать плоскость Нажмите S → X → 2, затем S → Y → 1.5 — придайте плоскости форму столешницы.
5 Сделать плоскость объёмной Выделите всю плоскость (A) → нажмите E → потяните вверх (например, на 0.2).
6 Разрезать объект по оси X Используем инструмент Loop Cut для создания разрезов на гранях столешницы. Ctrl + R → наведите на ось → После того как появится желтая линия нажмите ЛКМ, затем двигайте линию ближе с краю столешницы, для формирования будущих ножек стола. Проделайте это с двух сторон. Для удобства переключитесь на вид сверху(7)
7 Разрезать объект по оси Y Повторите действия п.6, но разрезайте по другой оси.
8 Выделить угловые грани Переключитесь в Face Select (3) → выделите четыре угловые грани снизу с помощью Shift.
9 Экструдировать ножки вниз Нажмите E — вытяните ножки вниз.
9.1 Удаляем лишние грани Переключитесь на вид сверху(7), выделите столешницу. Нажмите правую кнопку мыши - Dissolve Faces
10 Выделить столешницу Включите режим Toogle X-Ray в правой верхней части Viewport. Выделите верхнюю часть столешницы(режим отображения вершин)
11 Применить фаску к краям (Bevel) Ctrl + B → потяните мышью → прокрутите колесо мыши, чтобы добавить сегменты фаски.
12 Добавить обьекты Добавьте на стол дополнительные обьекты, например тарелки или кружки из предыдущей практической работы
13 Переключиться в рендеринг В правом верхнем углу 3D Viewport нажмите Z → Rendered, либо выберите иконку шарика.
14 Настроить камеру Shift + A → Camera, нажмите 0 на NumPad → G и R для позиционирования.
15 Сделать финальный рендер Нажмите F12 для рендера сцены.
16 Сохранить изображение В окне рендера: Image → Save As.



Ожидаемый результат:

  • Переключение между различными режимами выделения (вершины, рёбра, грани).
  • Перемещение, масштабирование и вращение элементов меша.
  • Использование инструментов Extrude, LoopCut, Bevel

Практическая работа №

Цель работы

Научиться использовать различные инструменты редактирования при моделировании обьекта в Blender

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Создайте новый проект и удалите куб Создайте новый файл: File → New → General. Удалите стартовый куб: выделите его → нажмите X → Delete.
2 Добавьте цилиндр Shift + A → Mesh → Cylinder.
3 Перейдите в режим редактирования Выделите цилиндр → нажмите клавишу Tab → Edit Mode.
4 Масштабируйте цилиндр по длине Нажмите S → Z → 2, чтобы вытянуть цилиндр по вертикальной оси (основа карандаша).
5 Создайте Loop Cut на цилиндре Используйте инструмент Loop Cut (Ctrl + R). Наведите на цилиндр и прокрутите колесико мыши, чтобы создать несколько горизонтальных разрезов (например, 3-4). Затем подтвердите разрезы кликом мыши (ЛКМ → ЛКМ).
6 Сформируйте наконечник карандаша Выделите верхнюю грань цилиндра (Face Select → выделите верхнюю грань). Нажмите E → вытяните немного вверх, затем нажмите S → уменьшите масштаб, формируя острую форму наконечника. Повторите эти действия несколько раз, постепенно уменьшая масштаб для создания заострённой формы.
7 Создайте срез карандаша снизу Выделите нижнюю грань, нажмите E → вытяните вниз совсем немного, затем S → немного уменьшите масштаб, чтобы сформировать область стержня карандаша.
8 Создайте Bevel (скругление) рёбер Перейдите в режим выделения рёбер (Edge Select), выделите рёбра на гранях наконечника карандаша и нижнего края, затем нажмите Ctrl + B и аккуратно двигайте мышью для скругления. Подтвердите действие кликом (ЛКМ).
9 Настройте освещение сцены Выйдите из режима редактирования (Tab), добавьте источник света: Shift + A → Light → Point. Переместите источник света вверх и чуть в сторону, чтобы карандаш был хорошо освещён.
10 Выполните рендер карандаша Перейдите во вкладку Render Properties и выберите движок рендера (Cycles или Eevee). Затем нажмите F12 для запуска рендера. Сохраните результат: Image → Save As.



Ожидаемый результат:

  • Переключение между различными режимами выделения (вершины, рёбра, грани).
  • Перемещение, масштабирование и вращение элементов меша.
  • Использование инструментов Extrude, LoopCut, Bevel
