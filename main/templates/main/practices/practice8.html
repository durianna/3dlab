Практическая работа: Инструмент Subdivide в Blender

Практическая работа №13

Цель работы

Научиться создавать органическую форму объекта с помощью инструментов Subdivide, Extrude, Inset, Bevel и пропорционального редактирования. Углубить понимание структуры меша и преобразования базовых примитивов.

Задание

Выполните следующие действия в среде Blender:

{% if is_mobile %}

Добавить сферу

Создайте новый проект с помощью File - New - General. Удалите куб. Добавьте сферу с помощью Shift + A → Mesh → UV Sphere

Перейти в режим редактирования

Выделите сферу → нажмите Tab чтобы перейти в режим редактирования обьектов

Применить Subdivide

Выделите всю сферу (A) → ПКМ → Subdivide → в панели настроек увеличьте число разрезов (Number of Cuts) до 2–3

Сформировать углубление под черенок

Для того, чтобы сделать углубление мы не будем вручную передвигать вершины, вместо этого воспользуемся инструментом Proportional Editing.Выделите верхнюю вершину → включите пропорциональное редактирование (английская буква O). После активации для удобства перейдите в вид спереди(1 на Numpud) и включите Toggle X-Ray. Чтобы переместить вершину вниз ровно нажмите G, а затем Z(чтобы зафиксировать перемещение по одной координате). Утопите вершину вниз с плавным переходом.

Сделать основание яблока

Аналогично шагу 4: выделите нижнюю вершину → пропорциональное редактирование → утопите вверх

Добавить отверстие для черенка

Выделите вершины в центре верхней части, и откорректировав область пропорционального редактирования(сделав ее меньше) сформируйте углубление под черенок(G → Z)

Добавить черенок

Черенок сформируем из цилиндра. На этом этапе отключим пропорциональное редактирование повторным нажатием чтобы не задевать яблоко. Shift + A → Mesh → Cylinder. Включите Toggle X-ray или каркасный вид чтобы отредактировать размер и положение цилиндра.

Моделируем черенок

Выделите верхние грани черенка, экстудируйте их так, чтобы получить изогнутую форму(E). Используя Scale измените толщину черенка чтобы он выглядел естесственно.

Моделируем лист

Добавим плоскость для листа Shift + A → Mesh → Plane

Моделируем форму листа

Выделите плоскость, разделите ее на большее количество полиготов с помощью ПКМ - Subdivide, измените количество разделений на 4 в параметрах.

Моделируем форму листа


С помощью Proportional Editing сформируйте форму листа и выгните его в разных местах.

Моделируем форму листа


Переместите лист к основанию яблока. С помощью Proportional Editing придайте неровности яблоку, чтобы оно выглядело ествественней.

Финальный рендер


Перейдите в обьектный режим(Tab). Переместите камеру и свет так, чтобы яблоко было хорошо видно в кадре. Нажмите правой кнопкой мыши по яблоку и выберите Shade Smooth для гладкого затенения. Нажмите F12 для рендера, сохраните результат.
