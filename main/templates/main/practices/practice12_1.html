Практическая работа: Создание базового материала в Blender

Практическая работа №20

Цель работы

Научиться создавать базовый материал для объекта, изменять его цвет, отражение и шероховатость в Blender.

{% if is_mobile %}

Создать новый проект

File → New → General. Удалить стартовый куб (X → Delete).

Добавить новый объект

Shift + A → Mesh → UV Sphere.

Перейти в режим Object Mode

Убедиться, что выбран режим Object Mode (Tab → Object Mode).

Открыть вкладку материалов

В правой панели Properties нажать иконку шара (Material Properties).

Добавить новый материал

Нажать кнопку New. Появится новый базовый материал.

Изменить цвет материала

В параметре Base Color выбрать любой новый цвет.

Настроить отражение

Параметр Metallic: установить значение 0.5 для среднего блеска.

Настроить шероховатость

Параметр Roughness: установить значение 0.3–0.4 для мягкого блеска.

Переключиться на Rendered View

В правом верхнем углу выбрать Rendered View (или нажать Z → Rendered).

Изменить цвет фона сцены

Вкладка World Properties → Color → выбрать подходящий фон.

Сделать рендер изображения


Настроить камеру → нажать F12 для рендера сцены. Поэспериментрируйте с различными настройками и обьектами, сохранив несколько вариантов рендера
