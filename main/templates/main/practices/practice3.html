Практическая работа: Настройки трансформаций, структура меша и параметры объекта в Blender

Изучение базовых трансформаций, структуры меша и свойств объектов в Blender.

Цель работы

Научиться выполнять базовые трансформации (перемещение, вращение, масштабирование), исследовать структуру меша, работать с режимами редактирования и изменять параметры объекта.

Задание

Выполните следующие действия в Blender:

№ Действие Метод выполнения
1Добавьте объект CubeShift + A → Mesh → Cube
2Переместите объектНажмите G и переместите объект мышкой или по осям (X, Y, Z)
3Поверните объектНажмите R → вращение вокруг выбранной оси
4Измените масштабНажмите S → масштабирование объекта
5Перейдите в режим редактированияTab → Edit Mode
6Выделите вершины, рёбра и граниИспользуйте клавиши 1 (Vertex), 2 (Edge), 3 (Face)
7Измените форму объектаВыделите и переместите/масштабируйте элементы меша
8Посмотрите статистику объектаВключите "Statistics" в Overlay → отобразится количество вершин и полигонов
9Откройте панель "Item"Нажмите клавишу N → вкладка "Item" → изучите параметры Transform
10Сбросьте трансформацииCtrl + A → Apply → Location/Rotation/Scale
11Измените имя объектаВ панели Properties → Object → поле Name
12Проверьте координаты объектаПанель "Item" → Location, Rotation, Scale
13Переключитесь в локальную/глобальную систему координатМеню вверху 3D-вида или клавиша "," и "."
14Установите точку вращения на центр массыPivot Point → Median Point / Individual Origins


Отчёт по работе

В отчёте учащийся должен:

  • Привести скриншоты трансформированных объектов до и после применения трансформаций.
  • Описание использованных режимов редактирования и что было изменено в структуре меша.
  • Указать параметры, отображённые в панели "Item" (координаты, масштаб и т.д.).
  • Рассказать, как он применял трансформации и зачем выполнялся сброс (Apply).

Дополнительно (по желанию)

  • Создайте несколько объектов и сгруппируйте их.
  • Попробуйте использовать snapping для точного позиционирования (включается кнопкой с магнитом).
  • Изучите вкладку Object Data Properties (зелёный треугольник) и найдите информацию о геометрии объекта.
