Практическая работа: Навигация в 3D-пространстве Blender

Практическая работа №4

Цель работы

Научиться выделять объекты с использованием различных методов (рамка выделения, лассо, круговое выделение) и применять трансформации (перемещение, вращение, масштабирование) с помощью горячих клавиш G, R и S. Использовать 3D- курсор для точного позиционирования объектов и изменения центра вращения с помощью различных вариантов Pivot Point. Применить инструмент Gizmo для трансформаций объектов.

Создание базовой сцены

Создайте новый проект. Добавьте на сцену три объекта: куб, цилиндр и сферу, и расположите их произвольно на сцене, используя изученные ранее методы трансформации (G , R, S). Например: Куб по центру, цилиндр слева, сфера справа.

Позиционирование камеры

В N-panel откройте вкладку Item. В разделе Location установите следующие значения: Location X = 2.0, Location Y = -3.0, Location Z = 1.0. Эти значения переместят куб в новые координаты в 3D-пространстве.

Изменить Rotation объекта через N-panel

Выделите камеру на сцене. Используя горячие клавиши перемещения (G) и вращения (R), разместите камеру таким образом, чтобы в её поле зрения попадали все три объекта одновременно. Воспользуйтесь комбинацией 0 (Numpad) для переключения вида из камеры.

Разделение рабочего пространства

Разделите экран на три отдельных окна, потянув за угол окна Blender. Для этого наведите курсор в верхний правый угол экрана, пока не появится иконка для разделения, и перетащите её. Это создаст дополнительные окна для различных видов.

Управление несколькими окнами

В каждом окне отдельно настройте масштаб и ракурс таким образом, чтобы удобно просматривать объекты сцены с разных точек. Например, в окне Top View настройте удобное отображение сцены сверху, а в камере и основном 3D-виде настройте обзор объектов спереди и сбоку. Чтобы вновь соединить окна в одно - ПКМ между двумя окнами - Join right/left

Создание дополнительной камеры

Добавьте еще одну камеру (Shift+A → Camera). Разместите новую камеру с другого угла сцены, чтобы получать альтернативный ракурс. Не забудьте о смене PivotPoint c 3D курсoра на Median Point, чтобы обьект снова пращался относительно себя. Переключайтесь между камерами с помощью сочетания Ctrl + 0 (Numpad) для того, чтобы сделать камеру активной(сначала ЛКМ по нужной камере, после ctrl + 0).

Управление несколькими окнами

Переключайтесь между двумя камерами (Ctrl + 0 (Numpad)) и убедитесь, что оба ракурса отображают разные перспективы сцены. Проверьте это во втором окне, настроенном на вид из камеры.

Сохранить проект

После выполнения всех шагов сохраните проект. Перейдите в меню File → Save As, выберите имя для файла, например, Simple_Scene.blend и нажмите Save As Blender File.
