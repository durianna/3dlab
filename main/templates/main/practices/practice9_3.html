Практическая работа: Использование модификаторов для моделирования обьектов. Boolean

Практическая работа №16

Цель работы

Создать модель шестерёнки с зубцами и центральным отверстием, используя примитивы и модификатор Boolean.

Задание

Выполните следующие действия в Blender:

{% if is_mobile %}

Создать новый проект и удалить куб

Откройте Blender → File → New → General. Выделите стартовый куб (ЛКМ) → нажмите X → Delete для удаления.

Добавить основной цилиндр для тела шестерёнки

Shift + A → Mesh → Cylinder. В появившемся меню (F9) установите параметры: Vertices: 64, Radius: 1, Depth: 0.4. Это основа шестерёнки.

Добавить вспомогательный цилиндр для зубца

Shift + A → Mesh → Cylinder. Установите параметры: Vertices: 32, Radius: 0.2, Depth: 0.6. Это заготовка для одного зубца.

Разместить вспомогательный цилиндр у края

Выделите маленький цилиндр. Нажмите G → X (или Y) и переместите его к краю основного цилиндра, чтобы он частично заходил внутрь. Сделайте это в режиме вида сверху(7)

Создать копии цилиндра по кругу



  1. Для того, чтобы все маленькие цилидры находились на одном расстоянии от центра основы мы можем воспользоваться PivotPoint. Так как 3D курсор по умолчанию находится в центре(Shift+C чтобы переместить) мы можем сменить пивот на 3D курсор чтобы все вращения проходили вокруг него. Смените пивот на 3D курсор в верхней части Viewport.
  2. Выделите маленький цилиндр (ЛКМ), который будет использоваться как зубец шестерёнки. Дублируйте его с помощью Shift + D
  3. Поверните новый цилиндр с помощью R на некотором расстоянии от первого "зубца". В левом нижнем углу откроется понель Duplicate object, в этой панели в графе Angle укажите значение 33
  4. Чобы не повторять все действия вручную вы можете повторить послеследнее сохраненное действие "Rotate" с помощью комбинации клавиш - SHIFT + R. Заполните окружность "зубцами" и откорректируйте их количество или расположение при необходимости.

Объединить все вспомогательные цилиндры


Для экономии времени обьединим все маленькие цилиндры в один обьект. Для этого воспользуемся инструментом Join. Выделите все вспомогательные цилиндры → Ctrl + J для объединения в один объект(или правой кнопкой мыши по обьектам - Join).

Добавить модификатор Boolean на основной цилиндр

Выделите основной цилиндр → Modifier Properties → Add Modifier → Boolean → Operation: Difference → выберите объединённые цилиндры в поле Object.

Применить модификатор и удалить цилиндры

Нажмите Apply в модификаторе. Затем выделите вспомогательные цилиндры → X → Delete.

Создать отверстие в центре

Shift + A → Mesh → Cylinder. Настроить Radius. Разместить в центре. Добавить новый модификатор Boolean → Difference → применить, удалить цилиндр.

Сгладить шестерёнку. Рендер.


ПКМ → Shade Smooth для сглаживания. Настроить камеру и свет, нажать F12 и сохранить рендер
