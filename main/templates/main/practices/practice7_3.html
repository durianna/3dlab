Дополнительные инструменты редактирования в Blender

Практическая работа №12

Цель работы

Сформировать модель стула с использованием базовых инструментов редактирования меша и точного позиционирования элементов.

Задание

Выполните следующие действия в среде Blender:

{% if is_mobile %}

Удалить стартовый куб

Выделите куб → нажмите X → выберите Delete.

Добавить плоскость (сидение)

Добавить плоскость (сидение)

Придать объём сидушке

Tab → Edit Mode → A → E → вытяните вверх на 0.1–0.2 ед.

Добавить разрез (Loop Cut)


Ctrl + R → прокрутите колесо → создайте 1 разрез и переместите ближе к краю. Таким же образом сделайте еще 3 разреза, у вас должны появится 4 угловые грани для будущих ножек стула.

Выделить зоны для ножек

Face Select (3) → выделите 4 угловые грани снизу

Экструдировать ножки

E → потянуть вниз на 1.5–2 ед.

Сделать фаски (Bevel)

Выделите вертикальные грани ножек → Ctrl + B → двигайте мышь

Добавить перекладину между ножками


Добавим новый обьект для перекладины. Shift + A → Add Cube. Затем масштабируем куб так чтобы он стал похож на перекладину стула с помощью → S, затем переместим его с помощью G → разместите между передними ножками

Включить Snap для выравнивания

Включите значок Snap (магнит) → режим Face или Vertex → выровняйте. Если вы случайно потеряли выделение с добавленного обьекта вы можете вернуть его нажав CTRL+Z, выделение в Blender так же считается действием. После использования выключите Snap повторным нажатием на магнитик.

Добавить вторую перекладину

Shift + D → переместить к задним ножкам

Создать спинку стула

Выделите 2 угловые грани сидения сверху, затем экструдируйте их → E вверх, это будет основа для спинки

Добавить перекладины для спинки

Shift + A → Cube → масштаб → переместить между опорами спинки, аналогично с перекладинами для ножек стула создайте перекладины для спинки. Здесь вы также можете использовать "магнит" для того, чтобы ровно разместить их.

Выполните рендер

Перейдите во вкладку Render Properties и выберите движок рендера (Cycles или Eevee). Затем нажмите F12 для запуска рендера. Сохраните результат: Image → Save As.
