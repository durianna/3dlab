Практическая работа: Моделирование простого объекта с использованием инструмента Extrude

Освоение инструмента выдавливания (Extrude) для моделирования сложной геометрии из примитивов.

Цель работы

Научиться использовать инструмент Extrude для создания новых геометрических форм на основе примитивов.

Задание

Выполните следующие действия в Blender:

№ Действие Метод выполнения
1Создание простого 3D-объекта (сферы или куба)Откройте Blender.Создайте новый проект. Удалите стандартный куб (если он есть). Добавьте сферу или куб через меню "Add" → "Mesh" → "Sphere" или "Cube".
2Добавление материала к объектуВыберите ваш объект (сферу или куб). Перейдите на вкладку "Material Properties" (с иконкой шара). Нажмите "New", чтобы создать новый материал. Настройте цвет материала, изменив параметр "Base Color".
3Добавление текстурыВ той же вкладке "Material Properties", найдите раздел "Surface". Нажмите на маленькую иконку рядом с "Base Color", чтобы открыть меню текстур. Выберите "Image Texture" из выпадающего списка. В появившемся окне выберите изображение текстуры (например, текстуру кирпичной стены или дерева). Примените текстуру к объекту.
4Настройка координат текстурыПерейдите в окно "UV Editing", чтобы развернуть объект и корректно наложить текстуру. Выберите объект, нажмите "U" для развёртки, затем выберите "Unwrap". Перейдите обратно в "Shading" для работы с текстурами.
5Настройка свойств текстурыВернитесь в "Material Properties" и подкорректируйте параметры текстуры (например, "Specular" для отражений или "Roughness" для матовости).
6Рендеринг сценыКогда все настроено, перейдите в окно "Render Properties". Выберите тип рендера (например, "Cycles" для более качественного изображения). Нажмите "F12" для рендеринга сцены.


Отчёт по работе

В отчёте учащийся должен:

  • Приложить изображение готового рендера сцены
  • Описать шаги, которые были выполнены при текстурировании
  • Отметить, какие текстуры или настройки использовались
  • Кратко описать, что удалось создать с помощью текстурирования.

Дополнительно (по желанию)

  • Экспериментируйте с различными типами текстур (например, с нормальными картами или с картами отражений).
  • Попробуйте изменить материалы объектов, чтобы увидеть, как влияет изменение различных настроек (например, металлическость или прозрачность).
