Практическая работа: Использование модификаторов для моделирования обьектов. Boolean

Практическая работа №14

Цель работы

Создать объёмный треугольный кусок сыра с помощью базовых примитивов и вырезать в нём отверстия с помощью сфер и модификатора Boolean.

Задание

Выполните следующие действия в среде Blender:

{% if is_mobile %}

Добавить плоскость

Создать новый проект, удалить стартовый куб и добавить плоскость. Shift + A → Mesh → Plane

Создать треугольник


Перейти в Edit Mode(Tab) → выделите 1 вершину в режиме редактирования вершин → нажмите X → Delete Vertices чтобы удалить одну вершину. Выделите две смежных вершины и обьекдините их с помощью Edge (F), чтобы получить треугольник.

Придать объём



Выделите все 3 точки, нажмите правой кнопкой мыши → New Edge/ Face from Vertices для того чтобы создать треугольную грань, затем вытяните ее чтобы сделать похожей по форме на кусочек сыра. Выделите грань → нажмите E → вытяните вверх (создаётся объём)

Придать объём

Добавим фаски чтобы сыр выглядел более обьемно. В режиме редактирования выделите все вершины кнопкой A, затем нажмите CTRL + B чтобы активировать инструмент Bevel(колесиком добавьте 2-3 уровня), создайте небольшой скос на гранях

Добавить сферу для будущего отверстия

Теперь нам необходимо добавить сферу, которой будем "вырезать" дырки для сыра. Сначала нам нужно перейти обратно в обьектный режим(Tab). Добавляем сферу: Shift + A → Mesh → UV Sphere → уменьшите масштаб → разместите частично внутри "сыра"

Скопировать сферу

Выделите кусок сыра → в панели справа нажмите на иконку гаечного ключа и добавьте модификатор Boolean(Add modifier → Generate → Boolean). В настройках выберите Difference(разница) → В поле для второго обьекта выберите сферу → нажмите Apply или CTRL+A

Применить модификатор Boolean

Переместите сферу на другое место, где хотите добавить отверстие и выполните аналогичные действия. Меняя положение и размер сферы добавьте несколько отверстий. Не забывайте, что модификатор Boolean в этом случае мы добавляем к модели сыра а не сферы.

Удалить сферы

После применения Boolean удалите сферу (X → Delete)

Сгладить объект

ПКМ по модели → Shade Smooth или Shade Auto Smooth.

Финальный рендер

Откорректируйте положение камеры и света. Нажмите F12 и сохраните результат рендера.
