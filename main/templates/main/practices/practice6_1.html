Практическая работа: Моделирование простого объекта с использованием инструмента Extrude

Освоение инструмента выдавливания (Extrude) для моделирования сложной геометрии из примитивов.

Практическая работа №8

Цель работы

Научиться моделировать простой контейнер с внутренним пространством и крышкой, используя выдавливание внутрь и точное позиционирование объектов.

Задание

Выполните следующие действия в среде Blender:

{% if is_mobile %}

Масштабировать куб (основа ящика)

Если куба на сцене нет — Shift + A → Mesh → Cube. Масштабируйте куб по оси Z на 0.5 единиц (S → Z → 0.5).

Войти в режим редактирования

Выделите куб → Tab → Edit Mode → включите режим выделения граней (Face Select → 3) → выберите верхнюю грань.

Inset верхней грани

Нажмите I (Inset) → переместите мышь внутрь, чтобы создать новую грань → ЛКМ для подтверждения.

Extrude внутрь

Нажмите E → потяните вниз, чтобы создать полость. Для удобства включите вид сбоку (Numpad 1) и режим X-Ray.

Сгладить края ящика

Выделите рёбра верхнего периметра → Ctrl + B → переместите мышь для фаски → ЛКМ для применения.

Создать крышку

Shift + A → Cube → масштабируйте по X/Y/Z → переместите G → Z → над ящиком. Вид сверху — Numpad 7.

Добавить ручку

Shift + A → Mesh → Cylinder → масштаб по Z и X → переместите ручку на центр крышки.
