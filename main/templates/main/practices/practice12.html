Практическая работа: Создание базового материала в Blender

Практическая работа №20

Цель работы

Научиться создавать базовый материал для объекта, изменять его цвет, отражение и шероховатость в Blender.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения Изображение
1 Создать новый проект File → New → General. Удалить стартовый куб (X → Delete).
2 Добавить новый объект Shift + A → Mesh → UV Sphere.
3 Перейти в режим Object Mode Убедиться, что выбран режим Object Mode (Tab → Object Mode).
4 Открыть вкладку материалов В правой панели Properties нажать иконку шара (Material Properties).
5 Добавить новый материал Нажать кнопку New. Появится новый базовый материал.
6 Изменить цвет материала В параметре Base Color выбрать любой новый цвет.
7 Настроить отражение Параметр Metallic: установить значение 0.5 для среднего блеска.
8 Настроить шероховатость Параметр Roughness: установить значение 0.3–0.4 для мягкого блеска.
9 Переключиться на Rendered View В правом верхнем углу выбрать Rendered View (или нажать Z → Rendered).
10 Изменить цвет фона сцены Вкладка World Properties → Color → выбрать подходящий фон.
11 Сделать рендер изображения Настроить камеру → нажать F12 для рендера сцены. Поэспериментрируйте с различными настройками и обьектами, сохранив несколько вариантов рендера



Ожидаемый результат:

  • Объект окрашен в выбранный цвет.
  • Корректно настроены параметры блеска и шероховатости.
  • Создан финальный рендер с новым материалом.

Вопросы для самоконтроля:

  1. Где в Blender создаются материалы для объектов?
  2. Как изменить цвет объекта через Base Color?
  3. Что регулируют параметры Specular и Roughness?

Практическая работа №21

Цель работы

Научиться назначать материалы объектам в Blender и настраивать их цветовые характеристики для создания законченного 3D-моделя бабочки.

Задание

Продолжая работу над моделью бабочки, выполните следующие действия:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Выделить объект "Тело бабочки" Откройте сохраненный бленд файл с моделью бабочки: File - Open. Перейдите в Object Mode (Tab) и выделите тело бабочки (Circle) в окне 3D Viewport или в списке объектов (Outliner).
2 Открыть панель материалов Перейдите в правую панель "Properties" и выберите вкладку с иконкой шара — "Material Properties".
3 Создать новый материал Нажмите кнопку New для создания нового материала. Появится стандартный материал с базовыми настройками.
4 Настроить цвет материала В разделе "Base Color" нажмите на цветной прямоугольник и выберите подходящий цвет для тела бабочки. Например, насыщенный оранжевый или жёлтый.Чтобы увидеть материалы в реальном времени, переключитесь в режим отображения Rendered: в правом верхнем углу окна 3D Viewport выберите соответствующую иконку (или нажмите клавишу Z → Rendered).
5 Добавить материал крыльям Выделите объект крыльев в Object Mode. Повторите аналогичные действия: вкладка материалов → New. Перейдем в Shader Editor. Разделите окно на 2 части потянув за левый нижний угол. Выберите тип окна - Shader Editor. Если вы добавили материал, то в окне Шейдера будет 2 ноды.
6 Материал для крыльев В окне ShaderEditor добавим текстуру для градиета Shift + A → Texture → Gradient Texture. И ноду для смешивания цветов ColorRamp (Shift + A → Converter → ColorRamp). Соедини: Gradient Texture → ColorRamp → Base Color у материала. В ColorRamp задай свои цвета для градиента.
7 Материал для глаз и усиков Ананлогичным образом добавьте материал для глаз и усиков, по желанию добавьте изученный модификатор Subdivision Surface для сглаживания крыльев бабочки. Поменяйте яркость и цвет мира в соотвествующей вкладке
8 Финальный рендер Настройте освещение и камеру и выполните финальный рендер нажатием клавиши F12.



Ожидаемый результат:

  • Тело бабочки окрашено в основной цвет.
  • Крылья имеют собственный яркий материал.
  • Глаза и усики отличаются своим цветом.

  2. Каким образом можно изменить цвет объекта?
  3. Как увидеть материал в окне 3D Viewport без рендера?
