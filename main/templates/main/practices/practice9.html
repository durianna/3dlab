Практическая работа: Использование модификаторов для моделирования обьектов. Boolean

Практическая работа №14

Цель работы

Создать объёмный треугольный кусок сыра с помощью базовых примитивов и вырезать в нём отверстия с помощью сфер и модификатора Boolean.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Добавить плоскость Создать новый проект, удалить стартовый куб и добавить плоскость. Shift + A → Mesh → Plane
2 Создать треугольник Перейти в Edit Mode(Tab) → выделите 1 вершину в режиме редактирования вершин → нажмите X → Delete Vertices чтобы удалить одну вершину. Выделите две смежных вершины и обьекдините их с помощью Edge (F), чтобы получить треугольник.
3 Придать объём Выделите все 3 точки, нажмите правой кнопкой мыши → New Edge/ Face from Vertices для того чтобы создать треугольную грань, затем вытяните ее чтобы сделать похожей по форме на кусочек сыра. Выделите грань → нажмите E → вытяните вверх (создаётся объём)
4 Придать объём Добавим фаски чтобы сыр выглядел более обьемно. В режиме редактирования выделите все вершины кнопкой A, затем нажмите CTRL + B чтобы активировать инструмент Bevel(колесиком добавьте 2-3 уровня), создайте небольшой скос на гранях
5 Добавить сферу для будущего отверстия Теперь нам необходимо добавить сферу, которой будем "вырезать" дырки для сыра. Сначала нам нужно перейти обратно в обьектный режим(Tab). Добавляем сферу: Shift + A → Mesh → UV Sphere → уменьшите масштаб → разместите частично внутри "сыра"
6 Скопировать сферу Выделите кусок сыра → в панели справа нажмите на иконку гаечного ключа и добавьте модификатор Boolean(Add modifier → Generate → Boolean). В настройках выберите Difference(разница) → В поле для второго обьекта выберите сферу → нажмите Apply или CTRL+A
7 Применить модификатор Boolean Переместите сферу на другое место, где хотите добавить отверстие и выполните аналогичные действия. Меняя положение и размер сферы добавьте несколько отверстий. Не забывайте, что модификатор Boolean в этом случае мы добавляем к модели сыра а не сферы.
8 Удалить сферы После применения Boolean удалите сферу (X → Delete)
9 Сгладить объект ПКМ по модели → Shade Smooth или Shade Auto Smooth.
10 Финальный рендер Откорректируйте положение камеры и света. Нажмите F12 и сохраните результат рендера.



Ожидаемый результат:

  • Объёмный треугольный кусок сыра с вырезанными отверстиями.
  • Использование Boolean Difference для вырезания с помощью сфер.

Вопросы для самоконтроля:

  1. Что делает модификатор Boolean в режиме Difference?
  2. Можно ли применить Boolean к нескольким объектам сразу?

Практическая работа №15

Цель работы

Научиться моделировать полую лабораторную колбу с горлышком и носиком, используя редактирование меша, трансформации и модификатор Boolean.

Задание

Выполните следующие действия в Blender шаг за шагом:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Удалить стартовый куб Выделите куб → нажмите X → Delete. Добавьте конус с помощью Shift + A → Mesh → Cone.
2 Редактировать конус (основание колбы) В левом нижнем углу нажмите на всплывающее меню "Add Cone". Установите:
  • Vertices: 64
  • Radius1: 1.2
  • Radius2: 0.3
  • Depth: 2.0
Если панель не появилась откройте ее с помощью F9. Затем нажмите S → Z → 0.7 для вертикального сжатия формы.
3 Сделать форму полой Чтобы сделать форму будущей колбы полой внутри воспользуемся модификатором Boolean. Создадим дубликат конуса с помощью Shift + D. Новую колбу сделаем чуть меньше основной и поместим внутрь основного конуса, ею мы будем "вырезать" полость. Выделите основную колбу, добавьте модификатор Boolean, выберите режим Difference. В качестве второго обьекта выберем Cone1(тот, что поменьше), в конце нажмите Apply и удалите второй конус.
4 Добавить цилиндр (горлышко) Добавим цилиндр для горлышка с помощью: Shift + A → Mesh → Cylinder. Затем масштабируем его с помощью S (масштаб) → 0.3 → S → Z → 2 и переместим: G → Z → разместите сверху колбы. Конец должен частично входить в верхнюю часть колбы.
5 Сделать горлышко полым Перейдите в Edit Mode → выделите верхнюю грань цилиндра → нажмите I (Inset) → E → выдавите внутрь, создавая внутреннюю стенку. Включите Toogle X-Ray и переключайтесь между видами для удобства на Numpad. Затем удалите внутренние нижние стенки цилиндра чтобы получить полую трубку.
6 Объединить части Вернитесь в обьектный режим(Tab). Выделите основание колбы → добавьте модификатор Boolean → выберите тип Union → в поле Object выберите цилиндр → нажмите Apply.
7 Удалить объект-горлышко Выделите цилиндр → нажмите X → Delete. Останется объединённая полая форма.
8 Сгладить колбу В режиме редактирования выделить донышко колбы и экструдировать его с помощью E. Затем добавьте фаски(Bevel) для обьема с помощью CTRL+B. Также эструдируйте горлышко колбы, полученные полигоны мастабируйте и сделайте чуть шире чем основание, чтобы добавить обьем. Вернитесь в обьектный режим. ПКМ по объекту → Shade Smooth
9 Рендер Настройте камеру и свет, нажмите F12 для рендера и сохраните результат



Ожидаемый результат:

  • Полая лабораторная колба, объединённая из конуса и цилиндра.
  • Внутренние стенки созданы с помощью Inset + Extrude.
  • Финальная форма сглажена и готова для дальнейшего оформления.

Вопросы для самопроверки:

  1. Как сделать объект полым?
  2. Какие операции выполнялись для объединения объектов?
  3. Для чего используется модификатор Bevel?

Практическая работа №16

Цель работы

Создать модель шестерёнки с зубцами и центральным отверстием, используя примитивы и модификатор Boolean.

Задание

Выполните следующие действия в Blender:

№ Действие Метод выполнения (инструкционная карта) Изображение
1 Создать новый проект и удалить куб Откройте Blender → File → New → General. Выделите стартовый куб (ЛКМ) → нажмите X → Delete для удаления.
2 Добавить основной цилиндр для тела шестерёнки Shift + A → Mesh → Cylinder. В появившемся меню (F9) установите параметры: Vertices: 64, Radius: 1, Depth: 0.4. Это основа шестерёнки.
3 Добавить вспомогательный цилиндр для зубца Shift + A → Mesh → Cylinder. Установите параметры: Vertices: 32, Radius: 0.2, Depth: 0.6. Это заготовка для одного зубца.
4 Разместить вспомогательный цилиндр у края Выделите маленький цилиндр. Нажмите G → X (или Y) и переместите его к краю основного цилиндра, чтобы он частично заходил внутрь. Сделайте это в режиме вида сверху(7)
5 Создать копии цилиндра по кругу
  1. Для того, чтобы все маленькие цилидры находились на одном расстоянии от центра основы мы можем воспользоваться PivotPoint. Так как 3D курсор по умолчанию находится в центре(Shift+C чтобы переместить) мы можем сменить пивот на 3D курсор чтобы все вращения проходили вокруг него. Смените пивот на 3D курсор в верхней части Viewport.
  2. Выделите маленький цилиндр (ЛКМ), который будет использоваться как зубец шестерёнки. Дублируйте его с помощью Shift + D
  3. Поверните новый цилиндр с помощью R на некотором расстоянии от первого "зубца". В левом нижнем углу откроется понель Duplicate object, в этой панели в графе Angle укажите значение 33
  4. Чобы не повторять все действия вручную вы можете повторить послеследнее сохраненное действие "Rotate" с помощью комбинации клавиш - SHIFT + R. Заполните окружность "зубцами" и откорректируйте их количество или расположение при необходимости.
6 Объединить все вспомогательные цилиндры Для экономии времени обьединим все маленькие цилиндры в один обьект. Для этого воспользуемся инструментом Join. Выделите все вспомогательные цилиндры → Ctrl + J для объединения в один объект(или правой кнопкой мыши по обьектам - Join).
7 Добавить модификатор Boolean на основной цилиндр Выделите основной цилиндр → Modifier Properties → Add Modifier → Boolean → Operation: Difference → выберите объединённые цилиндры в поле Object.
8 Применить модификатор и удалить цилиндры Нажмите Apply в модификаторе. Затем выделите вспомогательные цилиндры → X → Delete.
9 Создать отверстие в центре Shift + A → Mesh → Cylinder. Настроить Radius. Разместить в центре. Добавить новый модификатор Boolean → Difference → применить, удалить цилиндр.
10 Сгладить шестерёнку. Рендер. ПКМ → Shade Smooth для сглаживания. Настроить камеру и свет, нажать F12 и сохранить рендер



Ожидаемый результат:

  • Модель шестерёнки с зубцами, аккуратными скосами и центральным отверстием.
  • Освоение техники создания сложных объектов с использованием модификатора Boolean.

Вопросы для самопроверки:

  1. Какой режим модификатора Boolean использовался?
  2. Каким образом можно обьединить обьекты в один? В каких случах это может пригодится?
