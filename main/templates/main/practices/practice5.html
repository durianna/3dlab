Практическая работа: Моделирование простого объекта с использованием инструмента Extrude

Освоение инструмента выдавливания (Extrude) для моделирования сложной геометрии из примитивов.

Цель работы

Научиться использовать инструмент Extrude для создания новых геометрических форм на основе примитивов.

Задание

Выполните следующие действия в Blender:

№ Действие Метод выполнения Изображение
1 Удалить все объекты из сцены Нажмите A, чтобы выделить все объекты → нажмите X → выберите Delete.
2 Добавить цилиндр в сцену Нажмите Shift + A → Mesh → Cylinder.
3 Перейти в режим редактирования и выделить верхние вершины Выделите цилиндр → нажмите Tab для перехода в Edit Mode → нажмите Z или кликните по иконке Viewport Shading: Wireframe → включите режим выделения вершин (Vertex Select или клавиша 1) → выделите верхние вершины цилиндра рамкой.
4 Вставить (Inset) верхнюю часть цилиндра После выделения верхних вершин нажмите I (Inset) → переместите мышку, чтобы немного сместить вершины к центру → нажмите ЛКМ для подтверждения.
5 Вытянуть внутрь для создания полости С выделенной вставкой нажмите E (Extrude) → потяните мышку вниз по оси Z (или зажмите Z для ограничения по оси) → нажмите ЛКМ для завершения.
6 Расширить верхние вершины цилиндра Выделите все верхние вершины (рамкой или клавиша A в режиме вершинок) → нажмите S (Scale) → переместите мышку наружу для увеличения ширины верхней части цилиндра → ЛКМ для подтверждения.
7 Добавить тор и повернуть его по оси Y Нажмите Shift + A → Mesh → Torus → нажмите R, затем Y, затем введите 90 → нажмите Enter для поворота тора на 90° по оси Y.
8 Повернуть тор по оси Z на 90 С выделенным тором нажмите R, затем Z, затем введите 90 → нажмите Enter для подтверждения поворота.
9 Уменьшить тор и разместить его как ручку С выделенным тором нажмите S → уменьшите его размер → G для перемещения → переместите тор сбоку цилиндра, так чтобы он выглядел как ручка чашки → ЛКМ для фиксации.
10 Удалить лишние вершины ручки, выступающие за пределы чашки Перейдите в Edit Mode у тора → включите Wireframe View (Z → Wireframe) → выделите все вершины ручки, которые находятся за пределами чашки, с помощью рамки (B) → нажмите X → выберите Vertices для удаления.
11 Удалить лишние грани ручки, выступающие за пределы чашки В Edit Mode тора включите Wireframe View (Z → Wireframe) → выделите ненужные грани снаружи цилиндра → нажмите Delete (X) → выберите Faces для удаления выбранных граней.
12 Применить сглаживание к модели Перейдите в Object Mode (Tab) → нажмите A, чтобы выделить все объекты → правый клик мышью → выберите Shade Smooth.


Отчёт по работе

В отчёте учащийся должен:

  • Приложить скриншоты объектов до и после применения Extrude.
  • Описать шаги, которые были выполнены при моделировании.
  • Отметить, какие оси и значения использовались для Extrude и Scale.
  • Кратко описать, что удалось создать с помощью выдавливания.

Дополнительно (по желанию)

  • Создайте архитектурную деталь (например, колонну, проём в стене или лестницу).
  • Попробуйте использовать Extrude на сферах и цилиндрах.
  • Используйте инструмент Bevel после Extrude для сглаживания граней.
