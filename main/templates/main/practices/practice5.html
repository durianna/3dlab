Трансформации, структура меша и параметры объекта.

Практическая работа №6

Цель работы

Научиться переходить в режим редактирования (Edit Mode), понимать строение меша и взаимодействовать с его базовыми компонентами: вершинами, рёбрами и полигонами.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения Изображение
1 Вход в режим редактирования (Edit Mode) Откройте Blender и создайте новый проект. Стандартный куб уже выделен. Для перехода в режим редактирования:
- Нажмите клавишу Tab.
- Или в левом верхнем углу, где указано Object Mode, переключитесь на Edit Mode. Куб изменит вид, отображая теперь свои базовые элементы (вершины, рёбра, грани).
2 Изучение структуры меша (Mesh) Убедитесь, что вы в режиме редактирования (Edit Mode). Сверху слева найдите три иконки выбора режимов выделения:
- Vertex Select (выделение вершин)
- Edge Select (выделение рёбер)
- Face Select (выделение граней).
Переключитесь между режимами выделения и выделите элементы меша - вершины, ребра, грани.
3 Работа с вершинами (Vertices) Перейдите в режим выделения вершин (Vertex Select).
- Кликните ЛКМ на одной из вершин куба, чтобы выделить её.
- Переместите вершину с помощью клавиши G (grab). Закрепите позицию, нажав ЛКМ.
- Чтобы выделить несколько вершин, зажмите клавишу Shift и кликайте ЛКМ.
- Масштабируйте выбранные вершины с помощью клавиши S.
4 Работа с рёбрами (Edges) Перейдите в режим выделения рёбер (Edge Select).
- Выделите одно ребро, кликнув по нему.
- Поверните его, нажав клавишу R и двигая мышью.
- Выделите несколько рёбер, удерживая Shift, и попробуйте переместить их с помощью клавиши G.
5 Работа с гранями (Faces) Перейдите в режим выделения граней (Face Select).
- Выделите любую грань куба.
- Экструдируйте грань наружу: Нажмите клавишу E (extrude) и переместите курсор в нужное направление.
- Попробуйте удалить одну из граней: Выделите грань → нажмите X → выберите Faces.
6 Просмотр параметров объекта В режиме редактирования откройте боковую панель N-panel с помощью клавиши N. Перейдите на вкладку Item. Здесь можно вручную изменять параметры Location, Rotation, и Scale для выбранных элементов меша или всего объекта целиком. Попробуйте изменить значения для Location X, Y, Z и наблюдайте изменения.
7 Сохранение результатов Сохраните результат через меню Файл, отрендерите изображение настроив камеру и нажав F12.



Ожидаемый результат:

  • Переключение между различными режимами выделения (вершины, рёбра, грани).
  • Перемещение, масштабирование и вращение элементов меша.
  • Введение изменений в параметры объекта через боковую панель.

Вопросы для самоконтроля:

  1. Что такое меш в Blender, и какие основные элементы он включает?
  2. Как переключаться между различными режимами выделения (вершины, рёбра, грани)?
  3. Как работать с вершинами, рёбрами и гранями в режиме редактирования?

Практическая работа №7

Цель работы

Моделирование простого обьекта в режиме редактирования Blender

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения Изображение
1 Удалить все объекты из сцены Нажмите A, чтобы выделить все объекты → нажмите X → выберите Delete.
2 Добавить цилиндр в сцену Нажмите Shift + A → Mesh → Cylinder.
3 Перейти в режим редактирования и выделить верхние вершины Выделите цилиндр → нажмите Tab для перехода в Edit Mode → нажмите Z или кликните по иконке Viewport Shading: Wireframe → включите режим выделения вершин (Vertex Select или клавиша 1) → выделите верхние вершины цилиндра рамкой.
4 Вставить (Inset) верхнюю часть цилиндра После выделения верхних вершин нажмите I (Inset) → переместите мышку, чтобы немного сместить вершины к центру → нажмите ЛКМ для подтверждения.
5 Вытянуть внутрь для создания полости С выделенной вставкой нажмите E (Extrude) → потяните мышку вниз по оси Z (или зажмите Z для ограничения по оси) → нажмите ЛКМ для завершения.
6 Расширить верхние вершины цилиндра Выделите все верхние вершины (рамкой или клавиша A в режиме вершинок) → нажмите S (Scale) → переместите мышку наружу для увеличения ширины верхней части цилиндра → ЛКМ для подтверждения.
7 Добавить тор и повернуть его по оси Y Нажмите Shift + A → Mesh → Torus → нажмите R, затем Y, затем введите 90 → нажмите Enter для поворота тора на 90° по оси Y.
8 Повернуть тор по оси Z на 90 С выделенным тором нажмите R, затем Z, затем введите 90 → нажмите Enter для подтверждения поворота.
9 Уменьшить тор и разместить его как ручку С выделенным тором нажмите S → уменьшите его размер → G для перемещения → переместите тор сбоку цилиндра, так чтобы он выглядел как ручка чашки → ЛКМ для фиксации.
10 Удалить лишние вершины ручки, выступающие за пределы чашки Перейдите в Edit Mode у тора → включите Wireframe View (Z → Wireframe) → выделите все вершины ручки, которые находятся за пределами чашки, с помощью рамки (B) → нажмите X → выберите Vertices для удаления.
11 Удалить лишние грани ручки, выступающие за пределы чашки В Edit Mode тора включите Wireframe View (Z → Wireframe) → выделите ненужные грани снаружи цилиндра → нажмите Delete (X) → выберите Faces для удаления выбранных граней.
12 Применить сглаживание к модели Перейдите в Object Mode (Tab) → нажмите A, чтобы выделить все объекты → правый клик мышью → выберите Shade Smooth.


В отчёте учащийся должен:

  • Приложить скриншоты рендера объекта
  • Описать шаги, которые были выполнены при моделировании.
