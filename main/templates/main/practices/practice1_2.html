Практическая работа: Базовые трансформации обьекта

Практическая работа №2

Цель работы

Научиться добавлять новые объекты в сцену, а также изменять их масштаб, поворот и положение с помощью горячих клавиш S, R и G.

{% if is_mobile %}

Создайте новый проект

Перейдите во вкладку File - New - General чтобы создать новый проект

Переместить куб с помощью клавиши G

Выберите куб, нажмите G для активации инструмента перемещения. Переместите куб по оси X на -3.0. Чтобы перемещать объект по одной координате после нажатия G нажмите Х. Чтобы закрепить результат кликните ЛКМ..

Добавить новый объект (сфера)

Для добавления новой сферы нажмите Shift + A, выберите Mesh → UV Sphere. Сфера появится рядом с кубом.

Масштабировать сферу с помощью клавиши S

Выберите сферу, нажмите S для активации инструмента масштабирования. Масштабируйте сферу на S, 1.5, чтобы она была на 1.5 раза больше по всем осям.

Добавить новый объект (цилиндр)

Для добавления цилиндра нажмите Shift + A, выберите Mesh → Cylinder. Цилиндр появится в сцене.

Переместить цилиндр по оси Y на 3.0 с помощью G + Y

Выберите цилиндр, нажмите G + Y и переместите цилиндр по оси Y на 3.0.

Повернуть куб с помощью клавиши R

Выберите куб, нажмите R для активации инструмента поворота. Поворачивать обьект можно как и Scale относительно конкретной оси, так же можно после нажатия R, S или G ввести необходимые значения с клавиатуры. Поверните куб на 45° вокруг оси Z с помощью R - Z - 45.

Разместить сферу выше куба с помощью G + Z

Выберите сферу, нажмите G + Z и переместите её вверх на 2.0.

Сохранить проект

После выполнения всех шагов сохраните проект. Перейдите в меню File → Save As, выберите имя для файла, например, Simple_Scene.blend и нажмите Save As Blender File.
