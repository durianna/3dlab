Практическая работа №5

Цель работы

Научиться создавать базовую композицию сцены, настраивать рендер-движок, использовать и позиционировать камеру и источники света, а также получать изображения сцены с нескольких ракурсов..

Удалить куб из стартовой сцены

Выделите стандартный куб (по умолчанию уже выделен) → нажмите X → выберите Delete.

Добавить объект "обезьяна" (Suzanne)

Нажмите Shift + A → Mesh → Monkey → объект "Suzanne" появится в центре сцены.

Перейти в режим сглаживания для модели обезьяны

Выделите объект Suzanne → правый клик мышью → выберите Shade Smooth.

Перейти в рендерный вид сцены

В правом верхнем углу 3D Viewport нажмите иконку Rendered View (или клавиша Z → Rendered).

Установить рендер-движок на Cycles


Перейдите во вкладку Render Properties (значок камеры справа) → в поле Render Engine выберите Cycles (или Eevee для более быстрого рендера).

Добавить источник света

Light → переместите (G) и приподнимите свет над объектом (G → Z).

Установить первую камеру перед объектом

Выделите камеру → нажмите Ctrl + Alt + Numpad 0, чтобы переместить её на текущий вид → G и R для точной настройки позиции и угла.

Сделать рендер первого ракурса


Нажмите F12 для рендеринга изображения. После завершения → Image → Save As, чтобы сохранить рендер.

Переустановить камеру для второго ракурса (сбоку)


Переместите камеру сбоку от головы обезьяны (G → X или Y) → отрегулируйте угол обзора (R) → снова нажмите F12 для рендера второго изображения.

Изменить освещение для третьего кадра

Переместите свет вбок или сзади объекта.

Установить камеру для третьего ракурса (вид сверху или диагональный)


Переместите камеру в нужную позицию (например, под углом сверху) → скорректируйте вид → F12.

Ожидаемые результаты

  • Использовать сглаживание (Shade Smooth) для улучшения отображения объектов.
  • Настраивать рендер-движок (Cycles или Eevee) и просматривать сцену в режиме рендера.
  • Размещать и управлять источниками света.
  • Позиционировать камеру в разных местах сцены и выполнять рендеринг изображений с различных ракурсов.
