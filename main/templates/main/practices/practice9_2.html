Практическая работа: Использование модификаторов для моделирования обьектов. Boolean

Практическая работа №15

Цель работы

Научиться моделировать полую лабораторную колбу с горлышком и носиком, используя редактирование меша, трансформации и модификатор Boolean.

Задание

Выполните следующие действия в Blender шаг за шагом:

{% if is_mobile %}

Удалить стартовый куб

Выделите куб → нажмите X → Delete. Добавьте конус с помощью Shift + A → Mesh → Cone.

Редактировать конус (основание колбы)

В левом нижнем углу нажмите на всплывающее меню "Add Cone". Установите:

  • Vertices: 64
  • Radius1: 1.2
  • Radius2: 0.3
  • Depth: 2.0
Если панель не появилась откройте ее с помощью F9. Затем нажмите S → Z → 0.7 для вертикального сжатия формы.

Сделать форму полой



Чтобы сделать форму будущей колбы полой внутри воспользуемся модификатором Boolean. Создадим дубликат конуса с помощью Shift + D. Новую колбу сделаем чуть меньше основной и поместим внутрь основного конуса, ею мы будем "вырезать" полость. Выделите основную колбу, добавьте модификатор Boolean, выберите режим Difference. В качестве второго обьекта выберем Cone1(тот, что поменьше), в конце нажмите Apply и удалите второй конус.

Добавить цилиндр (горлышко)

Добавим цилиндр для горлышка с помощью: Shift + A → Mesh → Cylinder. Затем масштабируем его с помощью S (масштаб) → 0.3 → S → Z → 2 и переместим: G → Z → разместите сверху колбы. Конец должен частично входить в верхнюю часть колбы.

Сделать горлышко полым




Перейдите в Edit Mode → выделите верхнюю грань цилиндра → нажмите I (Inset) → E → выдавите внутрь, создавая внутреннюю стенку. Включите Toogle X-Ray и переключайтесь между видами для удобства на Numpad. Затем удалите внутренние нижние стенки цилиндра чтобы получить полую трубку.

Объединить части

Вернитесь в обьектный режим(Tab). Выделите основание колбы → добавьте модификатор Boolean → выберите тип Union → в поле Object выберите цилиндр → нажмите Apply.

Удалить объект-горлышко

Выделите цилиндр → нажмите X → Delete. Останется объединённая полая форма.

Сгладить колбу




В режиме редактирования выделить донышко колбы и экструдировать его с помощью E. Затем добавьте фаски(Bevel) для обьема с помощью CTRL+B. Также эструдируйте горлышко колбы, полученные полигоны мастабируйте и сделайте чуть шире чем основание, чтобы добавить обьем. Вернитесь в обьектный режим. ПКМ по объекту → Shade Smooth

Рендер

Настройте камеру и свет, нажмите F12 для рендера и сохраните результат
