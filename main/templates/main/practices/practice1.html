Цель работы

Научиться ориентироваться в интерфейсе Blender, выполнять базовые операции с объектами и использовать основные инструменты.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения
1 Развернуть окно 3D-вида на весь экран Навести курсор на 3D-вьюпорт, нажать Shift + Пробел.
2 Переместить 3D-курсор ЛКМ по нужной точке в 3D-пространстве.
3 Выделить камеру, свет и куб поочерёдно ПКМ по каждому объекту.
4 Множественное выделение Удерживать Shift и ПКМ по объектам.
5 Вид спереди, сбоку и из камеры Нажать 1, 3, 0 на NumPad.
6 Выделить куб ПКМ по кубу.
7 Переместить по X на -2 Перетянуть по красной оси с Ctrl.
8 Переместить по Y на 3 Перетянуть по зелёной оси вправо.
9 Переместить по Z на -1 Перетянуть по синей оси вниз.
10 Масштаб по X в 3 раза S → X, Ctrl + перемещение до 3.0.
11 Масштаб по Y в 1.3 раза S → Y, довести до 1.3.
12 Масштаб по Z в 0.5 раза S → Z, уменьшить до 0.5.
13 Повернуть по X на -15° N → ввести -15 в поле Rotation X.
14 Повернуть по Y на -10° Ввести -10 в поле Rotation Y.
15 Выполнить рендер сцены Нажать F12.
16 Сохранить итоговое изображение В рендер-окне нажать F3 → выбрать путь и формат.


Отчёт по работе

В отчёте учащийся должен:

  • Указать, какие объекты он добавил.
  • Какие трансформации применил.
  • В каком режиме рендеринга работал.
  • Приложить скриншот финального рендера.

Дополнительно (по желанию)

  • Примените к одному из объектов модификатор Subdivision Surface.
  • Попробуйте изменить цвет объектов во вкладке Materials.
