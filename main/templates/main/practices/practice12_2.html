Практическая работа: Создание базового материала в Blender

Практическая работа №21

Цель работы

Научиться назначать материалы объектам в Blender и настраивать их цветовые характеристики для создания законченного 3D-моделя бабочки.

Задание

Продолжая работу над моделью бабочки, выполните следующие действия:

{% if is_mobile %}

Выделить объект "Тело бабочки"

Откройте сохраненный бленд файл с моделью бабочки: File - Open. Перейдите в Object Mode (Tab) и выделите тело бабочки (Circle) в окне 3D Viewport или в списке объектов (Outliner).

Открыть панель материалов

Перейдите в правую панель "Properties" и выберите вкладку с иконкой шара — "Material Properties".

Создать новый материал

Нажмите кнопку New для создания нового материала. Появится стандартный материал с базовыми настройками.

Настроить цвет материала

В разделе "Base Color" нажмите на цветной прямоугольник и выберите подходящий цвет для тела бабочки. Например, насыщенный оранжевый или жёлтый.Чтобы увидеть материалы в реальном времени, переключитесь в режим отображения Rendered: в правом верхнем углу окна 3D Viewport выберите соответствующую иконку (или нажмите клавишу Z → Rendered).

Добавить материал крыльям

Выделите объект крыльев в Object Mode. Повторите аналогичные действия: вкладка материалов → New. Перейдем в Shader Editor. Разделите окно на 2 части потянув за левый нижний угол. Выберите тип окна - Shader Editor. Если вы добавили материал, то в окне Шейдера будет 2 ноды.

Материал для крыльев



В окне ShaderEditor добавим текстуру для градиета Shift + A → Texture → Gradient Texture. И ноду для смешивания цветов ColorRamp (Shift + A → Converter → ColorRamp). Соедини: Gradient Texture → ColorRamp → Base Color у материала. В ColorRamp задай свои цвета для градиента.

Материал для глаз и усиков



Ананлогичным образом добавьте материал для глаз и усиков, по желанию добавьте изученный модификатор Subdivision Surface для сглаживания крыльев бабочки. Поменяйте яркость и цвет мира в соотвествующей вкладке

Финальный рендер

Настройте освещение и камеру и выполните финальный рендер нажатием клавиши F12.
