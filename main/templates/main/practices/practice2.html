Практическая работа: Работа с окнами и навигацией в Blender

Научиться использовать окна, управлять интерфейсом, применять Gizmo и использовать навигационные инструменты.

Цель работы

Освоить работу с окнами Blender, использовать элементы интерфейса, Gizmo, горячие клавиши и различные способы навигации.

Задание

Выполните следующие действия в среде Blender:

№ Действие Метод выполнения
1Разделите окно 3D-вида на дваНаведите курсор на границу окна → ЛКМ и потяните вбок или вниз
2Измените тип одного из окон на TimelineВ левом верхнем углу окна выберите "Timeline" из выпадающего списка
3Объедините два окна обратно в одноПеретащите одну границу окна на другое, чтобы объединить
4Измените размер окна свойств (Properties)Наведите курсор на границу и потяните её влево или вправо
5Включите отображение GizmoВ правом верхнем углу 3D-вида → включите Move, Rotate, Scale
6Переместите объект с помощью GizmoПотяните за стрелку нужной оси (X, Y или Z)
7Измените центр вращения на 3D CursorВ верхнем меню 3D-вида выберите Pivot Point → 3D Cursor
8Вращение вида с помощью NumPadИспользуйте клавиши 1, 3, 7 и Ctrl + 1/3/7
9Переключитесь в ортографический видНажмите 5 на NumPad
10Центрируйте вид на объектеВыделите объект → нажмите точку (.) на NumPad
11Откройте главное меню и найдите пункт RenderВ верхней панели выберите Render → Render Image
12Откройте круговое менюНажмите клавишу Z или ~
13Удалите объект через контекстное менюПКМ по объекту → Delete
14Добавьте новый объектShift + A → Mesh → выберите, например, Cube
15Переместите новый объект клавишей GВыделите объект → нажмите G и переместите
16Сохраните текущий макет интерфейсаВ верхнем меню: Window → Save Screen Layout


Отчёт по работе

В отчёте учащийся должен:

  • Указать, какие окна он использовал (например, 3D Viewport, Timeline, Properties).
  • Какие действия выполнил с окнами (разделение, объединение, изменение размера).
  • Как использовал Gizmo и какие действия выполнял (перемещение, масштабирование и пр.).
  • Какие виды применял через NumPad.
  • Приложить скриншот интерфейса с изменённой конфигурацией окон.

Дополнительно (по желанию)

  • Измените цвет добавленного объекта во вкладке Materials.
  • Настройте собственное рабочее пространство и сохраните его как пресет.
  • Добавьте объект Text и попробуйте отредактировать его свойства в панели Properties.
