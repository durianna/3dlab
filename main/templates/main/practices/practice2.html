Практическая работа: Выделение и трансформация обьекта с помощью инструмента Gizmo

Практическая работа №3

Цель работы

Научиться выделять объекты с использованием различных методов (рамка выделения, лассо, круговое выделение) и применять трансформации (перемещение, вращение, масштабирование) с помощью горячих клавиш G, R и S. Использовать 3D- курсор для точного позиционирования объектов и изменения центра вращения с помощью различных вариантов Pivot Point. Применить инструмент Gizmo для трансформаций объектов.

Создание сцены

Создайте новый проект в Blender. Добавьте на сцену 4 объекта: куб, сферу, цилиндр и конус. Разместите их в разных местах сцены для удобства выделения. Используйте клавишу Shift + A для добавления объектов и клавишу G для перемещения.

Выделение объектов с помощью ЛКМ

Выделите куб, кликнув по нему левой кнопкой мыши. После этого выделите сферу, кликнув по ней. Все выделенные объекты подсветятся оранжевой рамкой.

Выделение объектов с помощью Shift + ЛКМ

Удерживая клавишу Shift, выделите одновременно куб и цилиндр, кликнув по ним левой кнопкой мыши. Оба объекта должны быть подсвечены.

Выделение с помощью Box Select (B)

Нажмите клавишу B для активации рамки выделения. Потяните рамку по объектам (например, кубу и сфере), чтобы выделить их.

Выделение с помощью Circle Select (C)

Нажмите клавишу C для активации кругового выделения. Используйте колесо мыши для изменения радиуса выделения и щелкайте по объектам, чтобы их выделить. Поверните выделенные обьекты нажав R. Выделенные обьекты поворачиваются так, будто это один обькт.

Выделение с помощью Lasso Select (Ctrl + ЛКМ)

Удерживая клавишу Ctrl, выделите объекты с помощью левой кнопки мыши, рисуя произвольную область вокруг объектов. Увеличьте все обьекты одновременно, нажав S после выделения обьектов.

Работа с 3D-курсор: позиционирование

Переместите 3D-курсор в центр группы обьектов с помощью комбинации Shift + ЛКМ. Чтобы вернуть 3Д курсор в центр комбинация: Shift + C.

Перемещение объектов относительно 3D-курсора

После того, как вы переместили 3Д курсор мы можем настроить PivotPoint так, чтобы обьекты трансформировались относительно 3Д курсора. Перейдите в настройки Pivot Point и выберите 3D курсор.

Работа с Pivot Point: использование 3D-курсора как PivotPoint

Теперь все обьекты будут трансформироваться(например вращаться) вокруг нового пивота - 3D курсора. Проверьте это выделив обьект и нажав R.

Изменение Pivot Point на "Individual Origins"

Перейдите в настройки Pivot Point и выберите Individual Origins. Переместите объекты (например, куб и цилиндр), чтобы увидеть, как каждый из объектов будет вращаться вокруг своей оси.

Возвращаем 3D курсор в центр

Чтобы вернуть 3Д курсор в центр комбинация: Shift + C

Включаем Гизмо

Если Gizmo не отображается, включите его вручную через настройки в панели 3D Viewport (нажмите на иконку Gizmo в верхнем правом углу).

Трансформации обьекта с помощью Гизмо

Потянув за оси Гизмо трансформируйте обьекты. Поверните их, масштабируйте.

Сохранить проект

После выполнения всех шагов сохраните проект. Перейдите в меню File → Save As, выберите имя для файла, например, Simple_Scene.blend и нажмите Save As Blender File.
